# Agent Workspace

This directory contains resources to extend and guide AI assistant behavior within this repository.

## Structure

- `/rules`: Markdown files defining specific coding standards or behavioral constraints for this project.
- `/skills`: Definitions of complex, multi-step capabilities that the AI can invoke.
- `/workflows`: Standardized procedures for common development tasks (e.g., deploy, code review).

## How to Add

- **Rule**: Create a `.md` file in `/rules` describing the constraint.
- **Skill**: Create a subfolder in `/skills` with a `SKILL.md` file.
- **Workflow**: Create a `.md` file in `/workflows` with YAML frontmatter and step-by-step instructions.
